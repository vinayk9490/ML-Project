## Project Overview

End-to-end supervised ML regression project that predicts a student's math score based on demographic and academic attributes (gender, race/ethnicity, parental education, lunch type, test preparation, reading score, writing score). Deployed as a Flask web app targeting AWS Elastic Beanstalk.

## Commands

**Install dependencies:**
```bash
pip install -r requirements.txt
```

**Run the training pipeline** (ingests data ŌåÆ transforms ŌåÆ trains 9 models ŌåÆ saves artifacts):
```bash
python src/components/data_ingestion.py
```

**Start the Flask web app** (serves prediction UI at `http://localhost:5000`):
```bash
python application.py
```

## Architecture

### Training Pipeline (triggered via `data_ingestion.py` `__main__`)

```
stud.csv ŌåÆ DataIngestion ŌåÆ DataTransformation ŌåÆ ModelTrainer
              (splits 80/20)   (fits preprocessor)   (trains 9 models w/ GridSearchCV)
                                      Ōåō                        Ōåō
                             artifacts/preprocessor.pkl   artifacts/model.pkl
```

- **[src/components/data_ingestion.py](src/components/data_ingestion.py)** ŌĆö reads `notebook/data/stud.csv`, writes `artifacts/{data,train,test}.csv`, then orchestrates the full pipeline in `__main__`
- **[src/components/data_transformation.py](src/components/data_transformation.py)** ŌĆö builds a `ColumnTransformer`: median imputation + StandardScaler for numericals, most-frequent imputation + OneHotEncoder + StandardScaler for categoricals; saves `preprocessor.pkl`
- **[src/components/model_trainer.py](src/components/model_trainer.py)** ŌĆö runs GridSearchCV (cv=3, n_jobs=-1) across 9 regressors (LinearRegression, DecisionTree, RandomForest, GradientBoosting, AdaBoost, KNN, XGBoost, CatBoost, LogisticRegression); selects best model by R┬▓ (minimum threshold 0.6); saves `model.pkl`
- **[src/utils.py](src/utils.py)** ŌĆö `save_object`/`load_object` (pickle wrappers), `evaluate_models` (GridSearchCV runner returning model scores dict)

### Inference / Web App

- **[application.py](application.py)** ŌĆö Flask app; `GET /predict` renders the form, `POST /predict` calls `PredictPipeline.predict()` and returns the score
- **[src/pipeline/predict_pipeline.py](src/pipeline/predict_pipeline.py)** ŌĆö `PredictPipeline.predict()` loads `preprocessor.pkl` + `model.pkl` from `artifacts/`, transforms the input DataFrame, returns prediction; `CustomData` wraps the 7 form fields into a DataFrame

### Cross-Cutting Concerns

- **[src/logger.py](src/logger.py)** ŌĆö configures dual logging (rotating timestamped file under `logs/` + terminal stdout)
- **[src/exception.py](src/exception.py)** ŌĆö `CustomException` captures filename and line number from `sys.exc_info()`; wrap all caught exceptions with this before re-raising
